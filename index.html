PORTFOLIO

    Smart House

    The Smart House project is a home automation system developed in C that enables communication and control of multiple devices within a smart home environment. The system uses pipes for inter-process communication to facilitate real-time interactions between different components.

    Network chat application

    The Network Chat Application is a multi-client chat system developed in C that supports real-time communication between users over a network. It uses linked lists to manage multiple client connections, threads for concurrent client servicing, and synchronization mechanisms to ensure thread safety. The application also includes file management for storing chat history.

    ATM system

    The ATM System is a desktop application developed in C# using Windows Forms (WinForms) to simulate the functionalities of an automated teller machine (ATM). The application provides a graphical user interface (GUI) for users to perform various banking transactions such as checking balances, withdrawing cash, and viewing transaction history.
